# Kelowna-Only Focus

Version 1 of the application is intentionally limited to Kelowna, British Columbia.

Reasons:

1. Simpler development
2. More relevant species lists
3. Easier validation of observations
4. Better user experience
5. Stronger local community engagement

The application should contain:

* Kelowna-focused species lists
* Kelowna-centered mapping
* Okanagan wildlife examples
* Kelowna landmarks and sample observations

Future expansion beyond Kelowna may occur but is not a Version 1 objective.
