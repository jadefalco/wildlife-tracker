# Version 1 MVP Features

## Required Features

### Mapping
- Interactive map
- Kelowna-focused map
- Wildlife sighting pins
- GPS centering

### Wildlife Reporting
- Species category selection
- Species selection
- GPS capture
- Date/time capture
- Photo upload
- Notes

### Submission
- Prepare Submission for Province
- Link to BC Wildlife Submission page

### Mobile
- Mobile-first design
- Large touch-friendly buttons

---

## Not Included in Version 1

- AI species identification
- User accounts
- User profiles
- Notifications
- Social sharing
- Provincial API integration
- Offline mode
