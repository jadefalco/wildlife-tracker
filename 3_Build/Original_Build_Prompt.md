# Original Build Prompt

## Initial Application Prompt

Build a mobile-friendly web application called "Kelowna Wildlife Tracker".

Purpose:

Allow residents of Kelowna, British Columbia to quickly record wildlife sightings using their mobile phones.

Core Features:

### Home Screen

* Interactive map centered on Kelowna, BC
* Wildlife sighting pins
* Large "Log Sighting" button

### Log Sighting

Users can submit:

* Species
* Photo
* Notes
* GPS location
* Date and time

### Confirmation

After submission:

* Show sighting details
* Add sighting pin to map

### Submission

Provide a "Prepare Submission for Province" option that directs users to official BC wildlife reporting resources.

### Design Requirements

* Mobile-first
* Clean and simple
* Nature-inspired colors
* Easy for casual users

### Version 1 Constraints

Do not include:

* User accounts
* AI species identification
* Social networking features
* Notifications
* Provincial API integration

Goal:

Reduce friction between wildlife observation and wildlife reporting.
