# Species Expansion Prompt

## Purpose

Upgrade the species selection process to improve scientific usefulness while remaining easy for users.

---

## Category Structure

Users first select:

* Bird
* Mammal
* Reptile / Amphibian

After selecting a category, users select a species.

The species dropdown should be searchable.
