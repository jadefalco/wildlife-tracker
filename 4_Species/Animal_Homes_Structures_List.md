# Animal Homes & Structures

> This file defines animal homes and structures that may eventually become a separate observation type. These are not wildlife species and should not currently be mixed into species lists.

## Purpose

Habitat features, animal homes, and structures that users may report separately from wildlife sightings.

These entries are intended for future use as a distinct observation type in Kelowna Wildlife Tracker.
