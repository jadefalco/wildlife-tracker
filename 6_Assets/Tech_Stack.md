# Technology Stack

## Purpose

This document defines the preferred technologies for Version 1.

The goal is to use modern, widely supported technologies while minimizing complexity.

---

# Frontend

Preferred:

* Next.js
* React
* TypeScript

Purpose:

* Mobile-friendly user interface
* Mapping interface
* Observation forms

---

# Styling

Preferred:

* Tailwind CSS

Purpose:

* Fast UI development
* Mobile responsiveness

---

# Mapping

Preferred:

* Leaflet
* OpenStreetMap

Purpose:

* Display wildlife observations
* GPS visualization

---

# Backend

Preferred:

* Next.js API Routes

Purpose:

* Simple backend architecture
* Minimal infrastructure

---

# Database

Preferred:

* Supabase PostgreSQL

Purpose:

* Managed database
* Easy integration
* Scalable
* Developer-friendly

---

# File Storage

Preferred:

* Supabase Storage

Purpose:

* Wildlife photo uploads

---

# Hosting

Potential Options:

* Vercel
* Netlify

Final hosting decision can be made later.

---

# Guiding Principle

Choose the simplest technology that successfully solves the problem.

Avoid unnecessary complexity.
